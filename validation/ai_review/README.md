# CVSS automated watcher/IA validation policy

This project will not depend on human/manual expert action for current progress.

## Current-project rule

The active validation path is automated watcher/IA validation. Human/manual expert adjudication is explicitly deferred to future work and must not block implementation, reporting, dashboard generation, or manuscript drafting.

## Naming

- validation/ai_review: automated watcher/IA validation used by this project.
- validation/expert_packet: optional future human expert packet retained for later comparative studies.
- validation/comparison_protocol: future comparison between automated watcher/IA and human review.

## Reporting language

Current article wording should describe the workflow as automated, watcher-mediated, evidence-linked validation. It should not claim independent human expert validation. Human comparison can be described as planned future work.

## Minimum automated evidence

Each automated validation row should include case_id, run_id, metric or decision target, automated value, watcher/IA judgment, confidence, rationale, and source artifact pointers.
